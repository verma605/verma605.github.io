AnnaPurti

AnnaPurti is a website made to resolve the problem of food wastage caused in hotels or during organising some events.

It acts as a platform between individuals or institutions with large amount of excess food and NGO's working for feeding millions of hungry people.

It was made as a concept website during final year of college. And can be used for real world implementation

Following are the features:

  • It is a dynamic web project made with Java Server Pages, which is know for its security.
  • The website is compatible to both mobile and desktop.
  • It uses MySQL for storage of data.
  • The frontend of the project is made with Bootstrap framework.
